# 💳 BNPL React (Buy Now, Pay Later UI)

A modern frontend application for a Buy Now, Pay Later (BNPL) system built with React. This project focuses on building a smooth user experience for installment-based payments, clean UI design, and scalable frontend architecture.

## 🚀 Features
- BNPL (Buy Now, Pay Later) flow UI
- Installment plan selection interface
- Dynamic payment breakdown
- Responsive and mobile-friendly design
- State management for payment flow
- API integration ready structure
- Clean and reusable components
- Modern UI interactions

## 🛠 Tech Stack
- React.js
- Vite
- JavaScript (ES6+)
- Redux Toolkit
- Tailwind CSS
- REST API Integration
- Axios
- React Router
- Lucide React Icons
- Git & GitHub

## 📌 Project Purpose
This project was built to understand and implement BNPL (Buy Now, Pay Later) user flows, payment installment logic, and scalable frontend architecture using React ecosystem tools.

## ⚙️ Core Modules
- 🛍 Product Selection UI
- 💰 Installment Plan Calculator UI
- 📄 Order Summary Page
- 🔐 Payment Flow Handling
- 📊 Payment Breakdown View
- 🔄 State Management with Redux
